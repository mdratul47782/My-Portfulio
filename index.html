Project #2

MovieHub: Your Ultimate Movie Database Experience

Explore the world of movies with MovieHub, a comprehensive movie database platform built using Next.js and integrated with the TMDB API. This project is designed to provide an engaging and interactive experience for movie enthusiasts while showcasing advanced features like custom API endpoints, server-side rendering, and MongoDB integration.

Features:

  • Server-Side Rendering: Efficiently renders movie lists and detail pages directly on the server.
  • Custom API Endpoints: All client-side network requests are routed through secure, custom API endpoints.
  • Dynamic Movie Details Page: Includes a "More Like This" section loaded using progressive rendering.
  • Compare Movies Page: Compare multiple movies in a dynamic and intuitive interface with search and selection capabilities.
  • Search Functionality: A global search bar to find and explore movies, with results displayed on a dedicated search results page.
  • User Management: Includes login and registration functionality with data stored securely in a MongoDB Atlas database.
  • Watch List Feature: Logged-in users can add movies to their watch list for later viewing.
  • Social Media Integration: Share movie details on social platforms with proper meta information.
  • Error Handling: Custom-designed error and not-found pages for a seamless user experience.

Next.js, Tailwind CSS, MongoDB Atlas, JavaScript

Tasker: Task Management System

Tasker is an intuitive task management system designed for everyone, simplifying task creation and organization. With Tasker, users can create and manage tasks with ease, leveraging features like:

  • Task Creation: Add tasks with details like title, description, tags, due date, priority, and owner.
  • Task Listing: View all tasks in an organized list.
  • Search with Debounce: Use keywords to search tasks quickly and accurately, with debounce for a smoother experience.
  • Task Categorization: Organize tasks by tag, priority, or owner for easier tracking.
  • Favorites: Mark important tasks as favorites for quick access.
  • Status Tracking: Track progress by updating task status as you work.
  • Edit Tasks: Make changes to tasks as needed.
  • Bulk Deletion: Delete one or more tasks efficiently.

Tasker brings powerful features together to simplify and enhance your task management experience.
