Project #1

Airbnb: A Seamless Hotel Booking Experience

Airbnb is a full-featured real estate booking platform built with Next.js and MongoDB. It offers a seamless experience for property listings, booking management, and secure payments. The platform ensures an optimized user experience through **server-side rendering** and **parallel routing**.

  • Server-Side Rendering: Fast loading property listings and details pages.
  • Authentication: Secure login and registration with JWT and Google OAuth.
  • Payment Processing: Users can securely book properties via Stripe.
  • Dynamic Reviews & Ratings: Only verified guests can leave feedback.
  • Interactive Calendar: Integrated date-picker for booking reservations.
  • Meta Information Optimization: Proper Open Graph meta tags for social media sharing.
  • PDF Receipt Generation: Downloadable booking receipts with all details.
  • Email Confirmation: Automated email receipt after successful booking.

Next.js, Tailwind CSS, MongoDB Atlas, Stripe API, JWT, React Hook Form
